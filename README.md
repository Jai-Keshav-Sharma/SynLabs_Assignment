# Synergy Labs Assignment Repository

This repository contains three completed AI automation assignments demonstrating practical applications of LLMs, LangGraph workflows, and generative AI.

### [`blog/`](blog/)
**SEO Blog Automation Tool** - An end-to-end AI pipeline that automatically scrapes trending electronics products, performs SEO keyword research, and generates optimized blog posts published to GitHub Pages.

**Key Technologies:** LangGraph, OpenAI GPT-4o-mini, DataForSEO API, SerpAPI, Jekyll  
**Live Demo:** [https://jai-keshav-sharma.github.io/seo-blog-automation/](https://jai-keshav-sharma.github.io/seo-blog-automation/)

---

### [`business/`](business/)
**Business Requirements ‚Üí Technical Specifications Pipeline** - An AI-powered system that converts plain-language business requirements into comprehensive technical documentation including system architecture, data schemas, and pseudo-code.

**Key Technologies:** LangGraph, OpenAI GPT-4o-mini, Groq Llama-3.3-70B, Pydantic  
**Workflow:** Analysis ‚Üí Module Decomposition ‚Üí Schema Design ‚Üí Pseudo-code Generation ‚Üí Documentation

---

### [`video/`](video/)
**Trending News ‚Üí AI Video Generator** - An automated pipeline that scrapes trending news, generates narration scripts using LLMs, creates visual content with Stable Diffusion XL, and composes 30-60 second videos.

**Key Technologies:** LangGraph, OpenAI, Stable Diffusion XL, MoviePy  
**Platform:** Google Colab  
**Notebook:** [View on Colab](https://colab.research.google.com/drive/1ELJjy4YAGdauBMeLTxYHjAMb1xRvWQzs?usp=sharing)
